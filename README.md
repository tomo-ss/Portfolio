# Portfolio

本リポジトリは、卒業制作「Monte Tavola」における
要件定義・設計・画面設計・デザインカンプ等の
制作資料をまとめたポートフォリオリポジトリです。

## フォルダ構成

- design/
  - デザインカンプ（PNG形式）
    - ユーザー画面
    - 管理者画面

- docs/
  - 要件定義：要件定義書（PDF）
  - 設計：基本設計書／詳細設計書／機能一覧／テーブル定義（PDF）
  - 画面設計：
    - 画面遷移図（PDF）
    - ワイヤーフレーム（PNG）
      - ユーザー画面
      - 管理者画面
  - テスト：簡易テストケース（PDF）

※ docs配下の各資料は、提出・閲覧用としてPDF形式で格納しており、  
あわせて編集用の元データ（Word / Excel）も保存しています。
