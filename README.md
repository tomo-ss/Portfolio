# Portfolio

本リポジトリは、卒業制作「Monte Tavola」における
要件定義・設計・画面設計・デザインカンプ等の
制作資料をまとめたポートフォリオリポジトリです。

## フォルダ構成

- design/
  - デザインカンプ（ユーザー画面・管理者画面）

- docs/
  - 要件定義：要件定義書
  - 設計：基本設計書／詳細設計書／機能一覧／テーブル定義
  - 画面設計：画面遷移図／ワイヤーフレーム
  - テスト：簡易テストケース
